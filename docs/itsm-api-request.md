# ITSM API — необходимые методы

## 1. Авторизация компании
```
POST /auth → токен, ID компании, название, ИНН, статус договора
```

## 2. Данные компании
```
GET /company/{id} → название, ИНН, контакты, статус договора, остаток часов
```

## 3. Сотрудники компании
```
GET /company/{id}/employees → [ID, ФИО, должность, email, телефон]
```

## 4. Заявки
```
GET /tickets?company_id=&status=&date_from=&date_to=
→ ID, организация, тема, статус, ответственный, даты, контакт
```

## 5. Передача оценки в заявку
```
POST /tickets/{id}/feedback → оценка, текст, дата, источник
```

## 6. Webhook (опционально)
```
POST на наш URL при создании / закрытии заявки
```
