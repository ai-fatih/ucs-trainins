# Быстрые ответы: Отчёты

> Шаблоны ответов для тикетов поддержки по формированию отчётов.

---

## 1. Отчёт не формируется / пустой

**Когда возникает:** Пользователь выбирает отчёт, но он не генерируется; показывает 0 строк.

**Ответ:**
1. Проверить период отчёта — выбранный диапазон не должен превышать 1 месяц (для некоторых отчётов)
2. Убедиться, что за выбранный период были операции
3. Проверить настройки фильтров (подразделение, категория)
4. Если отчёт по остаткам — выполнить расчёт остатков

---

## 2. Не сходится отчёт по выручке

**Когда возникает:** Пользователь некорректно сверяет данные; использует не тот отчёт для расхода блюд; путает общую выручку с выручкой по категориям.

**Ответ:**
Для сверки данных используйте:
- **Отчёт по расходу блюд** — для проверки количества проданных позиций
- **Отчёт по выручке** — для финансовых показателей
- **Отчёт по себестоимости** — для расчёта маржинальности
Не путайте общую выручку с выручкой по отдельным категориям.

---

## 3. Отчёт не печатается / не экспортируется

**Когда возникает:** Кнопка печати неактивна; экспорт в Excel не работает.

**Ответ:**
1. Проверить настройки принтера
2. Для экспорта — сохранить отчёт сначала на диск (CSV / Excel), затем открыть
3. Если кнопка неактивна — возможно, нужны права администратора
4. Перезагрузить клиента и попробовать снова

---

## 4. Некорректные данные в отчёте

**Когда возникает:** Цифры не совпадают с ожидаемыми; дублирование строк; расхождения с 1С.

**Ответ:**
1. Проверить, есть ли непроведённые документы за период
2. Если флаг «Расчётные остатки только по проведённым документам» не установлен — в расчёте участвуют непроведённые документы
3. Выполнить перерасчёт остатков
4. Расхождения с 1С — проверить настройки выгрузки и сопоставления товаров

---

## 5. Отчёт по алкогольной декларации

**Когда возникает:** Клиент пытается сформировать отчёт «Алкогольная декларация» в прежнем формате.

**Ответ:**
Отчёт «Алкогольная декларация» больше не сдаётся. Используйте отчёты в ЛК ФСРАР (ЕГАИС) или формируйте справки через запросы порционных остатков.

---

## 6. Как построить нужный отчёт (общая консультация)

**Когда возникает:** Пользователь не знает, какой отчёт выбрать для своей задачи.

**Ответ:**
Покажите пользователю путь: **«Меню» → «Отчёты» → выбор категории**.
Рекомендуйте:
- Для остатков — «Отчёт по остаткам товаров»
- Для продаж — «Отчёт по продажам / расходу блюд»
- Для себестоимости — «Калькуляция / отчёт по себестоимости»
- По ЕГАИС — запросы порционных остатков через **«Сервис» → «ЕГАИС» → «Запросы»**
