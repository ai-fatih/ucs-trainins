# Быстрые ответы: НДС

> Шаблоны ответов для тикетов поддержки по НДС (5%, 22%, 0%).

---

## 1. Смена НДС для блюда

**Когда возникает:** Нужно изменить ставку НДС для позиции меню (например, с 22% на 5% или с 5% на 0%).

**Ответ:**
1. Зайти в карточку блюда в RK7
2. Перейти на вкладку **«Налоги»**
3. Выбрать нужную ставку НДС из списка
4. Сохранить изменения
5. Выполнить проверку на кассе — пробить тестовый чек

**Важно:** Смена НДС влияет на выгрузку в 1С и на себестоимость.

---

## 2. НДС 5% — льготная ставка

**Когда возникает:** Вопросы про применение льготной ставки НДС 5% в общепите (п. 4 ст. 164 НК РФ).

**Ответ:**
Ставка НДС 5% применяется при одновременном соблюдении условий:
- Доля доходов от услуг общепита > 70%
- Среднемесячная численность сотрудников не превышает лимит
- Сумма доходов за год не превышает лимит (уточнить актуальные значения)
Если условия выполнены — ставку 5% можно установить в карточках блюд.

---

## 3. НДС 22% — повышенная ставка

**Когда возникает:** Вопрос про ставку 22% для определённых товаров.

**Ответ:**
Ставка НДС 22% применяется для отдельных видов товаров (если не подпадают под льготную ставку). Настройка — в карточке блюда (аналогично п. 1).

---

## 4. Способ учёта НДС в 1С

**Когда возникает:** Данные по НДС из StoreHouse некорректно передаются в 1С.

**Ответ:**
1. Проверить настройки способа учёта НДС в карточке товара StoreHouse
2. Проверить сопоставление ставок НДС в 1С и SH5
3. Если расхождение — может потребоваться обновление схемы выгрузки (создать заявку на обновление)

---

## 5. НДС при возврате

**Когда возникает:** Возврат товара из закрытой кассовой смены; НДС при возврате считается некорректно.

**Ответ:**
1. Возврат из закрытой кассовой смены выполняется через специальные документы
2. НДС пересчитывается автоматически на основе ставки, установленной в карточке
3. Если договор на поддержку отсутствует — подключение может быть платным (предупредить клиента)
