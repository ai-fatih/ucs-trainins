# Быстрые ответы: Выгрузка / Интеграция с 1С

> Шаблоны ответов для тикетов поддержки по выгрузке данных из StoreHouse в 1С.

---

## 1. Выгрузка в 1С не работает

**Когда возникает:** Данные из SH5 не передаются в 1С; выгрузка зависает; ошибка при экспорте.

**Ответ:**
1. Проверить настройки выгрузки в SH5: **«Сервис» → «Выгрузка в 1С»**
2. Убедиться, что указаны корректные реквизиты подключения (путь к общей папке, FTP)
3. Проверить, есть ли непроведённые документы за период
4. Если ошибка в 1С («не хватает товара на остатках» и т.п.) — ошибка со стороны 1С, нужен бухгалтер
5. Создать тикет в SD с описанием ошибки

---

## 2. Расхождения между SH5 и 1С

**Когда возникает:** Данные по количеству / сумме / себестоимости различаются между StoreHouse и 1С.

**Ответ:**
1. Проверить способ учёта НДС в карточке товара SH5
2. Отображение столбцов верное — количество в 1С равно сумме себестоимости из SH5
3. Если расхождение по конкретным товарам — проверить сопоставление номенклатуры
4. Рассмотреть обновление схемы выгрузки (создать заявку)

---

## 3. Консультация по вариантам выгрузки данных

**Когда возникает:** Пользователь хочет знать все возможные способы интеграции SH5 с 1С.

**Ответ:**
Варианты выгрузки из StoreHouse Pro в 1С:
1. **Прямая выгрузка** — встроенный механизм SH5 (через общую папку / FTP)
2. **Выгрузка через XML** — настраиваемый формат
3. **Через промежуточное ПО** — r_k интеграционные решения
4. **API** — для кастомных решений

Рекомендуем обратиться к документации или в отдел разработки для выбора оптимального варианта.

---

## 4. Ошибка 1С при расчёте Р/Н

**Когда возникает:** Ошибка «при расчёте Р/Н не хватает товара на остатках» в 1С.

**Ответ:**
Ошибка возникает на стороне 1С, когда расчётный счёт не может списать товар из-за нехватки остатков в учётной системе. Пути решения:
1. Проверить остатки товара в 1С
2. Сверить с фактическими остатками в SH5
3. Выполнить корректировку остатков в 1С (со стороны бухгалтера)
4. Если остатки в SH5 корректны — проблема в настройках обмена между системами

---

## 5. Выгрузка не запускается / завис компьютер

**Когда возникает:** Компьютер, на котором запускается выгрузка, завис; сессия прервана.

**Ответ:**
1. Перезагрузить компьютер
2. Убедиться, что службы SH5 запущены
3. Проверить, не заблокирован ли файл выгрузки другим процессом
4. Повторить выгрузку
5. Если зависание повторяется — создать тикет для техподдержки
