# Быстрые ответы: Маркировка / Честный Знак (ЧЗ)

> Шаблоны ответов для тикетов поддержки по маркированной продукции.
> Дополнение к `03_Регуляторика/ЧЗ/Пользовательская часть/08_FAQ.md`

---

## 1. Продажа воды без учёта ЧЗ (розлив в бокал)

**Когда возникает:** В заведении продают воду/напитки в розлив; не хотят сканировать каждую бутылку в ЧЗ.

**Ответ:**
В карточке блюда на вкладке **«Списание»** указать, что списание по ЧЗ не требуется. Тогда вода будет продаваться без сканирования марки. Если разливается в бокал — это не считается розничной продажей маркированного товара.

---

## 2. Остатки ЧЗ — где посмотреть

**Когда возникает:** Клиент хочет сверить остатки маркированной продукции с Честным Знаком.

**Ответ:**
Остатки по Честному Знаку можно посмотреть в **Личном кабинете Честного Знака** (lk.crpt.ru). Там отображаются все марки, поступившие на баланс организации, и их статус.

---

## 3. QR-код для ЧЗ

**Когда возникает:** Клиенту нужен QR-код для отображения в чеке или для маркировки товара.

**Ответ:**
QR-код для ЧЗ формируется автоматически при пробитии маркированного товара. Если он не печатается:
1. Проверить настройки шаблона чека в RK7 — должен быть включён вывод QR-кода
2. Если проблема остаётся — создать тикет в SD для отдела инсталляции

---

## 4. Консультация по ЛК ЧЗ

**Когда возникает:** Вопросы по работе с Личным кабинетом Честного Знака.

**Ответ:**
Предоставьте консультацию по работе с маркированной продукцией в общих чертах. По интерфейсу и функционалу ЛК ЧЗ — рекомендовать обратиться в техподдержку Честного Знака (crpt.ru/support), так как это внешняя система.

---

## 5. Ошибка при сканировании DataMatrix кода

**Когда возникает:** Кассир сканирует DataMatrix, но код не читается; ошибка «Марка не найдена».

**Ответ:**
1. Проверить, не сканируется ли код на русской раскладке клавиатуры
2. Убедиться, что марка сохранена в базе СХ (выполнить запрос порционных остатков по справке Б)
3. Если марка «серая» — уже списана; если «чёрная» — недействительна
4. Повреждённый код — использовать модификатор «Нет кода» в заказе

---

## 6. Максимум позиций с марками в чеке

**Когда возникает:** Чек не пробивается при большом количестве маркированных товаров.

**Ответ:**
Ограничение — **не более 128 маркированных позиций** в одном чеке. Если превышено — разделить заказ на несколько чеков.
