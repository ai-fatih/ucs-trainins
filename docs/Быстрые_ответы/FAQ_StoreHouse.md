# FAQ: Часто задаваемые вопросы по StoreHouse Pro

> Сводный FAQ на основе анализа ~5000 обращений в Service Desk.
> По каждому вопросу — краткий ответ и ссылка на подробный шаблон.

---

## 1. ЕГАИС / Алкоголь

### 1.1 Как настроить порционный алкоголь?
Необходимо включить порционную продажу в карточке блюда RK7 и настроить сопоставление товара ЕГАИС в SH5.
→ [Подробнее](Быстрые_ответы_ЕГАИС_алкоголь.md#1-порционный-алкоголь--настройка)

### 1.2 Почему не списывается алкоголь при продаже?
Марка может быть уже списана, не сохранена в базу, или к блюду не привязан комплект. Проверьте запрос порционных остатков по справке Б.
→ [Подробнее](Быстрые_ответы_ЕГАИС_алкоголь.md#2-порционный-алкоголь--марка-не-списывается--ошибка-при-продаже)

### 1.3 Как подтвердить накладную ЕГАИС?
Проверьте связь с УТМ, выполните сопоставление товаров. Если нет связи — восстановите соединение через QushSvc.exe.
→ [Подробнее](Быстрые_ответы_ЕГАИС_алкоголь.md#3-накладные-егаис--не-приходят--не-подтверждаются)

### 1.4 Ошибка при отправке акта списания?
Выполните запрос порционных остатков, проверьте наличие марок на остатке. Убедитесь, что марка отсканирована без лишних символов.
→ [Подробнее](Быстрые_ответы_ЕГАИС_алкоголь.md#4-акты-списания--ошибка-при-отправке)

### 1.5 Пропала кнопка пива на кассе?
Проверьте, не снят ли товар с крана (невидимые объекты). Привяжите GTIN на карточке блюда.
→ [Подробнее](Быстрые_ответы_ЕГАИС_алкоголь.md#5-пиво--розлив-кеги-кран)

### 1.6 Как оформить возврат товара ЕГАИС поставщику?
Создайте документ «Возврат товара поставщику» в SH5, укажите причину и отправьте через УТМ.
→ [Подробнее](Быстрые_ответы_ЕГАИС_алкоголь.md#6-возврат-товара-егаис)

## 2. Системные ошибки

### 2.1 Ошибка на кассе — что делать?
Проверьте карточку блюда (алкоголь, комплект, GTIN). Если алкоголь ведётся через DinB — обратитесь к ним.
→ [Подробнее](Быстрые_ответы_Системные_ошибки.md#1-ошибка-на-кассе)

### 2.2 Блюдо не отображается в списке?
Проверьте невидимые объекты, привязку GTIN, наличие крана (для пива), категорию блюда.
→ [Подробнее](Быстрые_ответы_Системные_ошибки.md#2-не-отображается-блюдо--кнопка-на-кассе)

### 2.3 Не работает сканер штрихкодов?
Убедитесь, что сканер настроен на DataMatrix, проверьте библиотеку ExtdllEGAIS.dll и драйверы COM-порта.
→ [Подробнее](Быстрые_ответы_Системные_ошибки.md#3-не-работает-сканер)

### 2.4 Завис StoreHouse — что делать?
Перезагрузите сервер, проверьте службу QushSvc.exe, посмотрите логи в QushMan.exe.
→ [Подробнее](Быстрые_ответы_Системные_ошибки.md#5-завис--не-отвечает-storehouse)

### 2.5 Не формируется Z-отчёт?
Проверьте подключение к принтеру, перезагрузите терминал. При необходимости напечатайте дубликат.
→ [Подробнее](Быстрые_ответы_Системные_ошибки.md#6-не-формируется-z-отчёт)

## 3. Отчёты

### 3.1 Отчёт не формируется / пустой?
Проверьте период (не более месяца), убедитесь, что были операции, проверьте фильтры.
→ [Подробнее](Быстрые_ответы_Отчеты.md#1-отчёт-не-формируется--пустой)

### 3.2 Данные в отчёте не сходятся?
Проверьте непроведённые документы, установите флаг «Расчётные остатки только по проведённым».
→ [Подробнее](Быстрые_ответы_Отчеты.md#4-некорректные-данные-в-отчёте)

### 3.3 Как правильно сверять выручку?
Используйте «Отчёт по расходу блюд» для количества и «Отчёт по выручке» для финансов.
→ [Подробнее](Быстрые_ответы_Отчеты.md#2-не-сходится-отчёт-по-выручке)

## 4. НДС

### 4.1 Как сменить ставку НДС для блюда?
В карточке блюда вкладка «Налоги» → выбрать нужную ставку → сохранить → проверить на кассе.
→ [Подробнее](Быстрые_ответы_НДС.md#1-смена-ндс-для-блюда)

### 4.2 Когда применяется НДС 5%?
При доле доходов от общепита > 70%, соблюдении лимитов по численности и сумме доходов (п. 4 ст. 164 НК РФ).
→ [Подробнее](Быстрые_ответы_НДС.md#2-ндс-5--льготная-ставка)

### 4.3 НДС некорректно передаётся в 1С?
Проверьте настройки способа учёта НДС в карточке товара SH5 и сопоставление ставок.
→ [Подробнее](Быстрые_ответы_НДС.md#4-способ-учёта-ндс-в-1с)

## 5. Скидки / Акции

### 5.1 Как создать новую скидку?
«Меню» → «Скидки» → создать → указать тип, размер, категории, период → проверить на кассе.
→ [Подробнее](Быстрые_ответы_Скидки_акции.md#1-создание-новой-скидки)

### 5.2 Скидка не применяется — почему?
Проверьте категории-ограничители, даты действия, временной лимит. Уберите лишние ограничения.
→ [Подробнее](Быстрые_ответы_Скидки_акции.md#2-скидка-не-отображается--не-применяется)

### 5.3 Как удалить акцию?
В списке акций → удалить (отключить). Для уже созданных заказов — удалить вручную.
→ [Подробнее](Быстрые_ответы_Скидки_акции.md#5-удаление-акции)

## 6. Маркировка / Честный Знак

### 6.1 Как продавать товары без сканирования марок (розлив)?
В карточке блюда на вкладке «Списание» указать, что списание по ЧЗ не требуется.
→ [Подробнее](Быстрые_ответы_Маркировка_ЧЗ.md#1-продажа-воды-без-учёта-чз-розлив-в-бокал)

### 6.2 Где посмотреть остатки ЧЗ?
В Личном кабинете Честного Знака (lk.crpt.ru).
→ [Подробнее](Быстрые_ответы_Маркировка_ЧЗ.md#2-остатки-чз--где-посмотреть)

### 6.3 Ошибка при сканировании DataMatrix?
Проверьте раскладку, сохранность марки в СХ, статус марки (не списана ли).
→ [Подробнее](Быстрые_ответы_Маркировка_ЧЗ.md#5-ошибка-при-сканировании-datamatrix-кода)

## 7. Выгрузка в 1С

### 7.1 Не работает выгрузка в 1С?
Проверьте настройки выгрузки в SH5, реквизиты подключения, наличие непроведённых документов.
→ [Подробнее](Быстрые_ответы_Выгрузка_1С.md#1-выгрузка-в-1с-не-работает)

### 7.2 Расхождения между SH5 и 1С?
Проверьте НДС, сопоставление номенклатуры. Количество в 1С равно сумме себестоимости из SH5.
→ [Подробнее](Быстрые_ответы_Выгрузка_1С.md#2-расхождения-между-sh5-и-1с)

### 7.3 Какие есть способы выгрузки?
Прямая выгрузка, через XML, через промежуточное ПО r_k, API.
→ [Подробнее](Быстрые_ответы_Выгрузка_1С.md#3-консультация-по-вариантам-выгрузки-данных)
