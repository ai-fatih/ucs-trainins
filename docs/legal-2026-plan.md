## Фаза 3 — Техническая фиксация согласий

**Backend (реализовано):**
- Модель `ConsentLog` в Prisma schema
- Модуль `ConsentModule` (controller + service)
- Endpoint `POST /consent` — принимает тип согласия, страницу, IP, user-agent
- Сохраняет в БД: id, consentType, page, ip, userAgent, grantedAt

**Frontend:**
- В `api-client.ts` добавлен `apiClient.consent.log()`
- Вызов заглушен через `.catch(() => ({ success: false }))` — не блокирует отправку формы

**Перспектива:**
- При реализации полноценной регистрации — вызывать `apiClient.consent.log()` при отправке формы
- Добавить `subjectName`, `contact` в ConsentLog при наличии авторизации
